# Cafax

"Cafax" es una aventura conversacional escrita con [fi.js](http://github.com/baltasarq/fi-js/). Se trata de una remake de la primera parte de "El enigma de Aceps", una aventura conversacional creada en los años 80 para Amstrad CPC.

Howard Carter acude tras la llamada de Lord Canarvon para explorar una pirámide que tiene intimidados a todos los trabajadores que se han adentrado en ella.

Puedes [jugar a Cafax on-line en icth.io](https://baltasarq.itch.io/cafax).
Léelo en: <a href="https://baltasarq.github.io/Cafax/">https://baltasarq.github.io/Cafax/</a>.
